# Example Blog Post

This is a **sample post** written in Markdown. You can add more `.md` files to the `public/blogs/` folder and list them in `index.json`.

## Features

- Headings, lists, and **bold** or *italic* text
- [Links](https://example.com)
- Code: `inline` and blocks:

```js
const greet = () => "Hello, world!";
```

## Adding a new post

1. Create a new `.md` file in `public/blogs/` (e.g. `my-post.md`).
2. Add an entry to `public/blogs/index.json` with `slug`, `title`, and `date`.
3. The post will appear in the Blogs section.
